Цей проект надає REST API для обробки тексту за допомогою бібліотеки NLTK.

## Встановлення

1. Клонувати репозиторій:
git clone https://github.com/Mir1a/test_project_for_Property_House.git

2. Створити віртуальне середовище та активувати його:
python -m venv venv
`venv\Scripts\activate`

3. Встановити залежності:
pip install -r requirements.txt

4. Для запуску сервера використовуйте команду:
uvicorn main:app --reload
